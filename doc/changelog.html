CHANGELOG (${date}):

  • New "Show Keys" button in Fullscreen map

  • New map option: "Show Entity Names" that lets you toggle name labels for pets, NPCs, etc.

  • Bugfixes for keybindings in 5.5.0b1

  • Fullscreen Map shows biome name under mouse in Singleplayer, even for unloaded chunks

  • Bugfix: Auto-Map wasn't using correct Biome to color grass and foliage.
