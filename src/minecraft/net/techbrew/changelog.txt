JourneyMap 2.0b4 for Minecraft 1.5 (March 20, 2013)
http://journeymap.techbrew.net

Requirements:

- Minecraft 1.5.1
- ModLoader or Forge

Changes since JourneyMap 2.0b3 for Minecraft 1.5:

- Recompiled for Minecraft 1.5.1
- Filename error inside zip fixed
- Ice color fixed
- Other players now displayed in multiplayer
- Some entity icons resized to be more proportional to the others
- JourneyMap has a new logo
- Added logo splash to web UI

Changes since JourneyMap 1.9.1 for Minecraft 1.4.6:

- New web UI appearance with added support for IE 9
- Faster chunk render cycles for improved game performance
- Faster web server functionality
- Faster web browser scripts
- Improved surface map accuracy and appearance
- Improved cave mapping accuracy and appearance
- Added the ability to toggle display of animals, villagers, pets, other players
- Mobs now show current heading on the map
- Removed blurring when zooming with web UI (except in IE 9)
- In-game UI now has options screen
- Replaced the embedded servlet engine with a modified version of the NIO webserver "Rupy" (http://code.google.com/p/rupy/)
- Internal work to allow for custom block mods
- The cave lighting property has been re-enabled
- Widened default mapping radius to 5 chunks (was 4)

Known issues:

- Translation updates needed for German, Russian
- Other players not displayed on multiplayer
- Bug (Tracking ID AW2-6EE-Z3QB): Maps overwrite on multi-world servers because the world seed is no longer provided by the MC server

Translations added: 

- None
