== What's Changed in JourneyMap Server 5.6 ==

* Implementation of the JourneyMap API 2.0-SNAPSHOT in JourneyMap Server now provides the means to control individual features of the JourneyMap client.  The API provides fine-grained control to mods on a per-user basis. '''Note: it is against the Minecraft EULA to require players to pay for features.'''

* The server configs in <code>minecraft/journeymap/server/5.6</code> provide server owners the means to control fine-grained features of the JourneyMap client depending on user role (player or Op) and Game Type (Creative, Survival, Adventure, and Spectator).

* There is a global <code>journeymap.server.op.global.config</code> file which defines what features are available to Ops for the various Game Types. By default this config applies to all dimensions.

* There is a global <code>journeymap.server.player.global.config</code> file which defines what features are available to regular players for the various Game Types. By default this config applies to all dimensions.

* There will be (initially-disabled) override file created for each dimension.  For example: <code>journeymap.server.op.dim0.config</code> and <code>journeymap.server.player.dim0.config</code>.  To enable the dimensional override files, set <code>"enabled":true</code> in each file as desired.

== Config File Options ==

The config file is located here: ''(server folder)/configs/JourneyMapServer/(world).cfg . ''The following options in the config file can be modified to suit your needs.  Changes require a server restart.
